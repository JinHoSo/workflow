# Getting Started with Workflow Engine

Welcome to Workflow Engine! This guide will help you get started with building powerful, extensible workflows.

## What is Workflow Engine?

Workflow Engine is a TypeScript-based workflow execution engine that allows you to create complex workflows by connecting nodes together. It provides:

- **Node-based architecture**: Build workflows by connecting reusable nodes
- **DAG execution**: Intelligent dependency resolution with parallel execution
- **Extensibility**: Create custom nodes and plugins
- **Type safety**: Full TypeScript support
- **Protocol-based design**: Consistent interfaces for execution, data flow, and error handling

## Prerequisites

Before you begin, make sure you have:

- **Node.js** >= 18.0.0
- **Yarn** package manager (or npm)
- Basic knowledge of **TypeScript** and **JavaScript**
- A code editor (VS Code recommended)
