# Integration Examples

Examples demonstrating integration with external services and APIs.

## Examples

### HTTP Integration

Make HTTP requests and process responses in workflows.

See: [http-integration.ts](./http-integration.ts)

### Database Integration

Connect to databases and perform operations in workflows.

See: [database-integration.ts](./database-integration.ts)

## Running Examples

1. Navigate to the example directory
2. Install dependencies: `yarn install`
3. Build: `yarn build`
4. Run: `node dist/example.js`

## Prerequisites

- Node.js >= 18.0.0
- Yarn package manager
- Workflow Engine installed
- Access to external services (for integration examples)
