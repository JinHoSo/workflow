# Advanced Examples

Advanced workflow examples demonstrating complex scenarios and custom implementations.

## Examples

### Custom Node

Create and use a custom node in a workflow.

See: [custom-node.ts](./custom-node.ts)

### Custom Plugin

Create and register a custom plugin with multiple nodes.

See: [custom-plugin.ts](./custom-plugin.ts)

## Running Examples

1. Navigate to the example directory
2. Install dependencies: `yarn install`
3. Build: `yarn build`
4. Run: `node dist/example.js`

## Prerequisites

- Node.js >= 18.0.0
- Yarn package manager
- Workflow Engine installed
- Understanding of custom node and plugin development
